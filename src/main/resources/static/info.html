Project Info

Foodie Go Delivery Application

Project Overview

Foodie Go is a web application designed to digitize food delivery services in Mongolia. This project was developed as a team effort under the iCode Academy Nippon Fall 2025 program.

Users can browse restaurants, order food, manage their cart, checkout, leave ratings and reviews. Restaurant owners can manage their menus, view orders, and update their restaurant information. The application features role-based access control for Customers, Restaurants, and Admins.

Foodie Go デリバリーアプリケーション

プロジェクト概要

Foodie Goは、モンゴル国のフードデリバリーサービスをデジタル化するために設計されたウェブアプリケーションです。このプロジェクトは、iCode AcademyのNippon Fall 2025プログラムの一環としてチームで開発されました。

ユーザーはレストランの閲覧、料理の注文、カート管理、チェックアウト、評価・レビューの投稿ができます。レストランオーナーはメニューの管理、注文の確認、レストラン情報の更新が可能です。このアプリケーションは、顧客、レストラン、管理者のロールベースのアクセス制御を備えています。
